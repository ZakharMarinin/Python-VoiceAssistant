# Voice Assistant Project

## Обзор проекта

Этот репозиторий был создан в рамках занятий для группы **Junior Python 513** - образовательный голосовой ассистент на Python.

---

## Что умеет делать

- **Распознаёт голосовые команды** с микрофона
- **Говорит в ответ** через встроенный синтезатор речи (офлайн)
- **Выполняет команды**: открывает сайты, показывает время, запускает приложения и т.д.
- **Понимает нечёткие запросы**: даже если ошибётесь в одной букве — ассистент поймёт

---

## 🔧 Технологии и библиотеки

| Библиотека | Назначение |
|-----------|------------|
| `SpeechRecognition` | Распознавание речи (микрофон → текст) |
| `pyttsx3` | Синтез речи (текст → голос, оффлайн) |
| `fuzzywuzzy` / `python-Levenshtein` | Нечёткий поиск команд |
| `num2words` | Чтение цифр вслух (время, числа) |
| `PyAudio` | Подключение к микрофону |

**Системные зависимости:**

- **Linux**: `espeak-ng`, `portaudio-devel`
- **Windows**: Python 3.12/3.13 для стабильной установки PyAudio
