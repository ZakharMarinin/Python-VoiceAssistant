## Функция `help me`

Ассистент должен уметь подсказать пользователю о своих умениях. После получения сообщения от пользователя по ключевый словам `help`, `help me` - бот должен перечислить какие команды умеет исполнять ассистент.

---
### Что нужно сделать?

**Шаг 1.  Регистрация команд**
Добавь в словарь `opts['cmds']` новую команду:

- `'help'`: триггеры — `('help', 'help me')`

---

**Шаг 2. Логика выполнения в `execute_cmd`**

Внутри функции `execute_cmd(cmd_key, raw_cmd)` добавь новое разветвление `elif cmd_key == 'help':`.

Внутри этого блока тебе необходимо:

1. Озвучить приветственное сообщение (например, _"I can help you with the following commands"_).
    
2. Напечатать в консоли понятный для человека список всех доступных функций.
    
3. Передать этот список в функцию `speak()`, чтобы ассистент его прочитал.

Пример вывода:
```
Vladimir help me
- I can help you with the following commands:
- time
- google
- youtube
- paint
- coin
- help
```

---
### Дополнительная информация

Перечисление команд **должно быть** динамическим. То есть нельзя просто заготовить строку с перечислением команд, бот должен сам проходить по нашему словарю и находить действующие команды. Это позволит, при добавлении новых команд, автоматически появляется при вызове `help`. 
Для выполнения можете использовать метод `keys()` и `join()`, чтобы собирать команды в единую форматированную строку.
