# num2words

Библиотека `num2words` - это библиотека способная переводить числа в понятные для TTS модели символы. 

[Документация к библиотеке](https://github.com/savoirfairelinux/num2words)

### 1. Установка

Для того чтобы установить данную библиотеку, выполните в терминале следующую команду:

```
pip install num2words
```

### 2. Как она устроена

`num2words` - это библиотека-конвертер. Она не имеет дело со звуком или микрофоном, она работает исключительно с типами данных Python (целыми числами, дробными, датами).

Её внутреннее устройство - это **огромный набор правил грамматики**. Она знает:

- Как склоняются числительные в разных языках (например, "один час", "два часа", "пять часов").
    
- Как правильно читать дробные числа, валюты и порядковые числительные (первый, второй, третий).
### 3. Главные возможности

В вашем проекте вы используете её для времени, но она умеет гораздо больше:

- **Перевод чисел:** `num2words(42)` → `"forty-two"`
    
- **Порядковые номера:** `num2words(42, to='ordinal')` → `"forty-second"`
    
- **Валюты:** `num2words(42, to='currency')` → `"forty-two euros"`
    
- **Поддержка языков:** Это её «киллер-фича». Вы можете просто указать параметр `lang='ru'`, и библиотека автоматически переключится на русскую грамматику.
### 4. Почему она критична для ассистента?

Компьютерные системы оперируют данными, а человек — смыслами.

- Если вы скажете `speak("15:30")`, синтезатор может прочитать это как "один пять двоеточие три ноль". Это звучит очень "по-роботски" и неприятно для уха.
    
- `num2words` проводит **нормализацию текста**. Она превращает сухие цифры в естественную фразу. Это делает вашего ассистента похожим на живого собеседника, а не на калькулятор.
